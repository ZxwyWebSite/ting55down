## ZxwyWebSite/ting55down
### 简介
+ 恋听网小说下载器

### 使用
+ 只需传入id参数，即可自动下载
+ 如 https://m.ting55.com/book/12526
+ 运行 `./ting55down -id 12526`

### 参数
+ -id {bookid}   #指定小说id
+ -v             #查看程序版本
+ -dp {downpath} #指定下载目录

### 注意
+ 网站有访问频率限制，下载一部小说大约要挂一晚上
+ 而且当你下载小说时，局域网内同公网ipv4的设备无法同时收听或下载
+ 当提示 `访问过快！` 时，请让程序等1小时，否则再次访问会重置等待时间
